Кратко (RU)

Можно: новые интерфейсы, фоновые задачи, свой HTTP-сервер (как internal_api), работа с ядром через NeyraAgent и конфиг.

Нельзя: править core/ из плагина; хранить ключи в yaml/коде; блокировать event loop без нужды; вызывать LLM мимо NeyraAgent для ответов ассистента.

Hello World (идея)

Папка interfaces/my_hello_plugin/, plugin.yaml с cli_modes: [hello], main.py с run_plugin(ctx). agent.chat — async → используйте asyncio.run. Если ctx.agent нет (не Discord) — NeyraAgent(ctx.config).

python main.py --mode hello

Short (EN)

Allowed: new transports, background workers, embedded servers (see internal_api), core integration via NeyraAgent and ctx.config.

Not allowed: modifying core/ from a plugin; secrets in yaml/source; blocking the asyncio loop without reason; bypassing NeyraAgent for assistant replies.

Hello World (idea)

Folder interfaces/my_hello_plugin/, plugin.yaml with cli_modes: [hello], main.py exporting run_plugin(ctx). Use asyncio.run for await agent.chat(...). If ctx.agent is None, use NeyraAgent(ctx.config).
