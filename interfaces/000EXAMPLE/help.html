Русский

Плагин — каталог interfaces/<папка>/ с plugin.yaml и модулем main_script. Ядро сканирует interfaces/*/plugin.yaml.

Структура (шаблон)

interfaces/000EXAMPLE/
  plugin.yaml
  HELP.md
  help.html
  core/main.py

Поля plugin.yaml

ПолеСмысл
idУникальный идентификатор.
enabledВключён ли плагин в рантайме.
lifecycleresident / on_demand.
cli_modesСписок имён для python main.py --mode …; можно оставить пустым.
main_scriptПуть к Python от корня плагина.

Контракт

В main_script нужна функция:

def run_plugin(ctx: PluginContext) -> None:
    ...

ctx.root — корень проекта, ctx.config — конфиг, ctx.agent — агент или None (см. core/plugin_sdk.py).

Примеры в репозитории

interfaces/discord_text/, interfaces/internal_api/.

English

A plugin is a folder under interfaces/<name>/ with plugin.yaml and the main_script module. The core globs interfaces/*/plugin.yaml.

Layout (template)

interfaces/000EXAMPLE/
  plugin.yaml
  HELP.md
  help.html
  core/main.py

plugin.yaml fields

FieldMeaning
idUnique id.
enabledRuntime on/off.
lifecycleresident / on_demand.
cli_modesNames for python main.py --mode …; may be empty.
main_scriptPath to the entry Python file.

Contract

Export from main_script:

def run_plugin(ctx: PluginContext) -> None:
    ...

See core/plugin_sdk.py for PluginContext.

Reference plugins

interfaces/discord_text/, interfaces/internal_api/.
