# Публикация плагинов

## Минимум для публикации
- Полная папка `interfaces/<id>/`.
- `plugin.yaml`, `main.py`, `config.example.yaml`.
- README по установке и env переменным.

## Рекомендации
- Не публикуйте `config.yaml` с локальными данными.
- Добавьте smoke-команду проверки.
- Опишите ограничения безопасности и необходимые права.
